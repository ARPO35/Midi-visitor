# 配置栏层级设计

本文档记录配置栏重组后的层级。导入 / 清除和播放控制属于高频操作，不折叠，固定显示在配置栏顶部；其余可配置项按内容和类别折叠，默认全部收起，方便快速浏览。

## 顶部常驻区域

| 区域 | 内容 | 默认状态 |
| --- | --- | --- |
| 导入 / 清除 | MIDI 文件导入、外部音频导入、清除外部音频、配置包导入 / 导出 | 始终展开 |
| 播放控制 | 播放 / 暂停、回到开头、播放进度跳转 | 始终展开；仅在已加载 MIDI 轨道后显示 |

## 默认折叠分类

| 一级分类 | 用途 | 默认状态 |
| --- | --- | --- |
| 播放参数 | 启动延迟和输出音量 | 折叠 |
| 同步与音乐 | BPM、移调、外部音频对齐参数 | 折叠 |
| 轨道过滤 | 单独显示或隐藏 MIDI 轨道 | 折叠 |
| 音符与滚动 | 音符样式、滚动方向、速度、音高轴布局 | 折叠 |
| 画布与取景 | 应用背景和视口安全边距 | 折叠 |
| 窗口外观 | 可视化窗口的背景、模糊、边框和阴影 | 折叠 |
| 波形叠加 | 外部音频波形显示、模式、线条和峰值精度 | 折叠 |

## 交互规则

- 导入 / 清除和播放控制不折叠，保持顶部可见。
- 其余一级分类默认折叠。
- 展开一个分类不会自动关闭其他分类。
- 条件显示的控件仍按用途归类，不按当前数据源临时移动。
- `VisualConfig` 的字段名和取值语义保持不变。
- `stretch` 保持现有行为：修改音高间距时按比例同步调整 `offset`。
